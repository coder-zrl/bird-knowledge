# Taxonomy

## Preferred Types

- `concept`: abstract idea, mental model, claim, distinction.
- `entity`: person, organization, tool, project, artifact.
- `reference`: source-backed factual note, paper, article, spec, post.
- `skill`: reusable procedure or way of working.
- `synthesis`: higher-level analysis connecting multiple notes.
- `journal`: time-bound record or session note.
- `project`: active knowledge project.
- `instruction`: repository or agent instruction.
- `taxonomy`: local metadata convention.
- `note`: general note when a more specific type is not useful yet.

## Tag Style

- Use lowercase kebab-case tags.
- Keep tags sparse.
- Prefer links for relationships and tags for broad filtering.
