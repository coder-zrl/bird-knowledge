# 人生重置的一日协议

人生重置的一日协议是一种短周期反思方法：用一天时间把默认生活从自动驾驶中拉出来，明确不想继续的生活、想要建设的方向，以及明天就能执行的小行动。

## Structure

Morning:

- 写出 anti-vision：如果什么都不变，未来会变成什么样。
- 写出 vision：如果三年后的生活理想，它大概是什么结构。
- 找出旧身份：当前行为到底在保护什么。

Daytime:

- 定时观察自己在逃避什么。
- 记录什么时候更有生命感。
- 识别哪些行为是在选择安全，而不是建设。

Evening:

- 总结真正卡住自己的内部模式。
- 写下一年方向、一个月项目、明天的 2-3 个行动。

## Bird Knowledge Application

这个协议可以用于个人知识库建设：

- Anti-vision: 继续收藏材料但不形成自己的模型。
- Vision: 形成一个能被自己和 Codex 共同使用的长期知识系统。
- One-month project: 完成一本书的 OKF 化阅读。
- Tomorrow: 读一小节，写一个 reference update 或 concept note。
