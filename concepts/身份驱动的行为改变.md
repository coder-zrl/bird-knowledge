# 身份驱动的行为改变

身份驱动的行为改变认为：一个人很难长期维持与自我叙事冲突的行为。

如果旧身份仍然完整，新的行动就会一直像外部压力；如果身份和目标结构改变，新的行动会更容易变成“我就是这样的人会做的事”。

## Model

```text
identity -> goals -> attention -> behavior -> feedback -> identity
```

身份不是一个静态标签，而是由目标、注意力、行为和反馈构成的循环。改变行为时，真正困难的部分往往不是行动本身，而是这个循环会保护原来的自洽性。

## Use

用这个概念检查一个改变计划：

- 我想要的行为属于什么样的人？
- 我当前的行为在保护什么旧身份？
- 新身份今天能通过哪个微行为被证明？
- 这个微行为如何与 [福格行为模型](/references/福格行为模型.md) 中的动机、能力和提示结合？
